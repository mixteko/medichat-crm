# MediChat CRM

Prototipo inicial de CRM medico para gestionar conversaciones tipo WhatsApp, agenda de citas, pacientes, tienda medica pequena y flujos basicos de chatbot.

## Modulos incluidos

- Panel clinico con resumen operativo.
- Inbox tipo WhatsApp con respuestas rapidas y verificacion de citas.
- Agenda medica con confirmacion y reprogramacion.
- Directorio ligero de pacientes.
- Tienda medica con carrito, stock y registro de pedidos.
- Flujos demo de bot para agenda, verificacion de citas y tienda.

## Como abrirlo

Abre `index.html` en el navegador. La aplicacion no requiere instalacion ni dependencias externas.

## Siguiente etapa sugerida

- Backend con autenticacion y roles.
- Base de datos para pacientes, citas, productos y conversaciones.
- Integracion oficial con WhatsApp Business Cloud API.
